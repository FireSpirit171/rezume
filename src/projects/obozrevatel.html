Описание

Сайт был разработан для личного пользования. Ввиду частого просмотра фильмов и сериалов мне понадобился удобный инструмент для написания обзоров на них и дальнейшего их хранения.

Для данного веб-сайта были использованы следующие технологии:

Фронтенд

  • HTML - сайт написан на 3-ёх HTML-шаблонах
  • CSS - стили для сайта укомплектованы в один style.css

Бэкенд

  • Go - бэкенд сервиса написан на Go с использованием бибилиотеки Gorilla Mux.
  • MySQL - обзоры на просмотренные фильмы хранятся в MySQL и доступны для редактирования.
