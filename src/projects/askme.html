Описание

Проект создан в процессе обучения в образовательном центре VK при МГТУ им. Н. Э. Баумана. Целью работы являлось создание сайта вопросов-ответов по аналогии с Stack Overflow. MVP функционал сайта - регистрация/авторизация, создание постов, оставление комментариев, а также оценки правильности комментариев (лайки/дизлайки).

Для данного веб-сайта были использованы следующие технологии:

Фронтенд

  • HTML5 - сайт написан с помощью возможностей шаблонизатора Django
  • Bootstrap - разработка разметки и стилей велась с помощью популярной библиотеки стилей Bootstrap
  • JavaScript - для добавления технологии AJAX и обеспечения взаимодействия интерфейса с пользователем
  • Nginx - HTTP-сервер использовался для раздачи статики и кэширования данных

Бэкенд

  • Django - бэкенд сервиса написан на Django с использованием популярных библиотек.
  • PostgreSQL - база данных для хранения всей информации.
  • Gunicorn - для взаимодействия Nginx и Python был настроен WSGI-сервер.

Процесс разработки

Процесс разработки проекта был разделен на 6 основных этапов:

  • Верстка статического сайта. На этом этапе были созданы основные HTML-шаблоны для будущего сайта.
  • Обработка HTTP-запросов. Были созданы основные view, настроена маршрутизация, а также внедрено постраничное отображение (пагинация) содержимого сайта.
  • Проектирование модели данных. На этом этапе была спроектирована модель базы данных. Созданная БД была наполнена большим количеством вопросов (> 100 000) и ответов (> 1 000 000). Эти данные были настроены для отображения на сайте.
  • Авторизация и обработка форм. После настройки базы данных была подключена встроенная система авторизации Django и разработаны основные формы добавления данных
  • Загрузка картинок и обработка AJAX запросов. На этом этапе был добавлен функционал аватарок (загрузка картинок) и настроена обработка AJAX запросов (лайки).
  • Настройка веб-серверов. На финальном этапе был настроен Gunicorn и написан wsgi-скрипт для запуска приложения. Затем был настроен Nginx для раздачи статического контента и проксирования запросов.
