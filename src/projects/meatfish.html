Описание

Проект создан на курсе "Разработка интенрнет-приложений". Продуктовые требования проекта были строго регламентированы. Среди них - возможность просмотра услуг, поиск по названию или по ценовому диапазону, оформление заявки зарегистрированным пользователем. Далее сформированная заявка отправлялась на проверку модератором. При успешном прхождении проверки заказ оформлялся а пользователю выдавался сгенерированный системой qr-код. Референс для дизайна сайта был взят с ресторана FRANK by Баста

Для данного веб-сайта были использованы следующие технологии:

Фронтенд

  • React/Next.js - фронтенд сайта написан с помощью библиотеки пользовательских интерфейсов React, а позже переписан на Next.js для добавления статической генерации.
  • SASS - препроцессор SASS был выбран ввиду его удобного синтаксиса, а также встроенного инструмента миксинов.
  • Vite - разработка проекта велась с использованием локального сервера Vite.
  • GitHub Pages - весь фронтенд с использованием mock-объектов был развернут на GitHub Pages.

Бэкенд

  • Django REST framework - бэкенд написан с помощью DRF в соответствии с подходом REST API.
  • PostgreSQL - база данных для хранения всей информации.
  • Redis - сессии пользователей хранятся в Redis.

Архитектура

Previous Next

Проект развернут на виртуальной машине Ubuntu 22.04. На нём запущены S3-хранилище Minio, PostgreSQL для хранения информации о услугах и заявках, и Redis для хранения сессий пользователей. Бэкенд и фронтеэнд работают на 8000 и 3000 портах соответственно. Кроме того для создания десктопного приложения проект был развернут на Tauri v2.0. Фронтенд сервиса был дополнительно развернут на mock-объектах на github-pages.
